# lhl-final

## Backend setup

- gem install bundler (for Bundler 2)
- bundler install

## Client setup

- npm install

## Ruby Version:
`2.3.5`

## Rails Version:
`5.0.7`


## Start backend server:
navigate to /backend
$ rails s -p 3001 -b 0.0.0.0

## database name (development):
backend_test_final

## Start frontend server
navigate to /client
$ npm start
